# World Cup Trip Timer

Countdown site to June 11, 2026 (USA flight). Vanilla JS + Vite. Deployed to GitHub Pages.

## Stack
- Vanilla JS + Vite
- Plain CSS (no framework)
- GitHub Actions → GitHub Pages

## Agents
- `agents/design.md` — visual design
- `agents/development.md` — JS/CSS implementation
- `agents/devops.md` — CI/CD pipeline
- `agents/pr-review.md` — code review

## Commands
```
npm run dev      # local dev
npm run build    # production build → dist/
npm run preview  # preview build
```

## Rules
- No emojis in UI
- No extra dependencies
- Model: claude-sonnet-4-6
