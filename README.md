# World Cup Trip Timer

Countdown site to June 11, 2026 — USA road trip, World Cup 2026, Las Vegas, Miami, Bahamas cruise.

## Live Site

https://lieltavorxyz.github.io/world-cup-trip-timer

## Stack

- Vanilla JS + Vite
- Plain CSS
- GitHub Actions → GitHub Pages

## Local Dev

```bash
npm install
npm run dev
```

## Deploy

Push to `main` → GitHub Actions builds and deploys automatically.
