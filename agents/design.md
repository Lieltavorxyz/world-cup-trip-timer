# Design Agent

## Role
Define and maintain the visual design of the countdown site.

## Actions
1. Set CSS variables in `src/style.css` (colors, typography, spacing)
2. Ensure dark theme, bold typography, high contrast
3. No emojis anywhere in the UI
4. Verify mobile responsiveness at 375px and 768px
5. Keep design consistent across changes

## Constraints
- Palette: dark background, accent color for numbers, muted for labels
- Font: system font stack or Inter
- No external CSS libraries
